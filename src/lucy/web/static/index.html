Información de la I.A

Capacidades técnicas

  • PLN multilingüe (es/en), detección de idioma y análisis semántico
  • Normalización y lematización con NLTK cuando aplica
  • Clasificación de intenciones con modelo TensorFlow y fallback heurístico
  • Sistema de plugins y servicios externos con hooks seguros
  • Memoria a largo plazo (embeddings e índice) para recuperación contextual
  • Logging estructurado y métricas de rendimiento
  • Autocompletado y normalización avanzada de entradas
  • Configuración dinámica en runtime y perfiles de usuario

Incluye validaciones en tiempo real para formularios y saneamiento de entradas. El motor adapta la respuesta según el idioma y el contexto, con umbrales de confianza configurables.

Arquitectura

Usuario → LucyAI → Intención → Respuesta
   ├─ ConversationDB
   ├─ LoggingSystem
   └─ ConfigManager
  • Plugins con `PluginManager` y contratos seguros
  • Servicios externos desacoplados con clientes y reintentos
  • Memoria semántica con índice configurable
  • Web/API con FastAPI y protección CSRF
  • ConfigManager observa cambios en `config/config.json` (watcher)
  • Integración híbrida opcional con Django para administración

La configuración dinámica observa cambios en tiempo real y los aplica sin reiniciar. La base de datos optimiza consultas con índices por usuario y fecha.

Especificaciones de rendimiento

  • Objetivo p95 de respuesta < 2s en escenarios comunes
  • Umbral de confianza configurable y cache de resultados
  • Persistencia eficiente con SQLite y índices dedicados
  • Métricas de latencia y cobertura de pruebas disponibles
  • Optimización de repintados y uso moderado de efectos visuales
  • Cache de clasificación y contexto reutilizable por sesión

El sistema registra métricas clave (latencia, aciertos de intención, carga de CPU) y permite auditoría para mejorar continuamente el desempeño.

Casos de uso

  • Asistente personal local con historial y contexto
  • Integración de comandos `!api`, `!nlp`, `!mem` para extensiones
  • Laboratorio de PLN y prototipado rápido
  • Monitoreo y auditoría para entornos educativos
  • Prototipado de flujos conversacionales con trazabilidad y métricas

La interfaz web facilita flujos guiados y atajos rápidos desde el sidebar para acceder a la documentación y funcionalidades clave.

Futuras funcionalidades

Se planean mejoras de accesibilidad, localización y soporte de nuevas fuentes de datos con permisos granulados.

Derechos de uso

Uso personal y educativo. Consulta el README para condiciones adicionales.

Seguridad

  • CSRF con token y cookie de sesión `HttpOnly`
  • Hash PBKDF2-HMAC-SHA256 con salt único por usuario
  • Sanitización básica de inputs y comparación constante
  • Rate limiting y CORS configurables

Se recomienda mantener secretos fuera del repositorio y revisar logs estructurados para auditoría.

Accesibilidad

  • Contraste y tipografía conforme a WCAG AA
  • Navegación por teclado en enlaces y formularios
  • Layout responsive con contenedores semitransparentes
  • Scroll suave con preferencia de usuario respetada

La interfaz prioriza legibilidad y consistencia visual, con elementos de interacción claros y estados visibles.
